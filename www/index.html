list.it -- before you forget it!

list.it, the Latitudinal Information Scrap-Trapper that Indexes Things - is a small, simple note-keeping tool for solving a big, complex task -- helping you manage all of the little bits of information you need to keep track of each day .

list.it does this by focusing on speed and simplicity. We have gotten rid of everything except a way to get things in and out quickly, so that you can get things out of your head and somewhere you can access easily any time.

features

  • Simple Browser-Sidebar interface

    list.it lives in your Firefox browser sidebar, which can be pulled up and pulled away using a customizable hot key.

    At the top is a search box and a note input box; under that are your notes.

  • Hotkeys and quick input box

    One of the harder problems we're facing. Not only do we have to find a good hotkey that isn't already taken on most people's machines, but then we need to get them working across the 3 major platforms (Mac/Windows/Linux). This involves writing a native component for each platform. This is theoretically possible and we're still working on the actual implementation.

  • Synchronize across computers

    Looking to support synchronization across multiple computers. Currently, emax is working on setting up a django server to help with this. As soon as the server is up and I have an API, the logic will be mostly on my side and should be fairly easy to set up.
