list.it -- before you forget it!

list.it, the Latitudinal Information Scrap-Trapper that Indexes Things - is a simple note taking tool with a grand name -- and a grand goal. list.it aims to help you quickly capture and stash away all the random bits of information you generate each day, so you can get back to them effectively and easily.

Our design aims for simplicity, speed and elegance. Building up on our previous ethnographic work on information scraps...blah blah

    One of the harder problems we're facing. Not only do we have to find a good hotkey that isn't already taken on most people's machines, but then we need to get them working across the 3 major platforms (Mac/Windows/Linux). This involves writing a native component for each platform. This is theoretically possible and we're still working on the actual implementation.
