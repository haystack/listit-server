list.it .. before you forget it!

an easy-to-use note taking tool for Firefox

Here are some of the things we're working on:

Features
  • Runs in your browser sidebar
    Lorem ipsum dolor
  • Use it whether you're online or off
    Lorem ipsum dolor
  • Synchronize across computers

    Looking to support synchronization across multiple computers. Currently, emax is working on setting up a django server to help with this. As soon as the server is up and I have an API, the logic will be mostly on my side and should be fairly easy to set up.

  • Pop-up the input box to quickly add a note

    One of the harder problems we're facing. Not only do we have to find a good hotkey that isn't already taken on most people's machines, but then we need to get them working across the 3 major platforms (Mac/Windows/Linux). This involves writing a native component for each platform. This is theoretically possible and we're still working on the actual implementation.

  • Wiki syntax

    I like the functionality of the interface I have now, but with some tweaking of colors and stylesheets, I could make it much nicer. Also, I may want to pull it out of the sidebar and into its own chrome window so it doesn't take up too much of my browsing space.
